ABOUT

This website was created for the final project of "Tecnologias Web" for the course of "Tecnologias de Comunicação Multimédia" from ISMAI.

The function of this website is to help people find new movies to watch, dividing them by categories based on their genre. It also has information about the production of the movie, like the director, main cast and release year, so that they can research about it if they are interested. It also has a link for the trailer and some pictures from the movie itself.

Project made by Movie Time, composed of Gonçalo Cortez nº 36409 and Ricardo Morais nº 37258.
